About

📘 Student Ledger

Student Ledger is a simple and user-friendly student registration system designed to manage student records efficiently. It allows users to add, update, and remove student details in an organized format.

🚀 Key Features

  • Add new students quickly
  • Update existing student details
  • Delete students instantly
  • Dynamic student list rendering
  • Clean and interactive interface
  • Responsive for all devices

🛠 How to Use

➕ Add Student

Fill in student details and click Add Student.

✏ Update Student

Click Edit, modify details, then click Update.

❌ Remove Student

Click the Cross (X) button to delete a student.
